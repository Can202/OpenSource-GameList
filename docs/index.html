Open Source Game List

  • Minetest - voxel game engine like Minecraft
  • Stepmania - dance and rhythm game like Dance Dance Revolution
  • Quaver - competitive rhythm game like Osu Mania
  • Unknown Horizons - 2D realtime strategy simulation with an emphasis on economy and city building
  • Rigs of Rods - vehicle simulator that uses soft-body physics to simulate the motion destruction and deformation of vehicles
  • FlightGear - a sophisticated and professional Flight Simulator
